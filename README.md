# 开始使用

`pipx install graiax-cli` (推荐，防止污染全局环境)

或者

`pip install graiax-cli`

## 新建项目

`graiax init` 交互式初始化一个 `Graia Framework` 项目 (Partly WIP)

## 管理插件 (WIP)

`graiax plugin new` 新建插件

`graiax plugin list` 查看插件列表

`graiax plugin add` 添加插件
